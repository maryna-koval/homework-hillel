1. SyntaxError - виникає при синтаксичній помилці в будь якому синтаксісі в програмі, неправильно написані
методи, загублені дужки, двокрапки і тд. Приклад(загублена двокрапка після умови з if):
a = 3
if a = 3
    print(a)

2. TypeError - помилка яка виникає під час віконання певної операції з об'єктом неправильного типу. Наприклад
оператора "+" з об'єктами типу string.
a = 5
b = 'Hello world'
c = a + b

3. ZeroDivisionError - помилка яка виникає при спробі ділення на 0.
x = 4
y = 0
z = x / y

4. ValueError - виникає коли в функції використовується аргумент неправильного типу або значення. Наприклад коли
користувач намагається перетворити рядок в ціле число.
x = int("abc")

5. NameError - помилка яка виникає коли програма на Python не знаходить змінну з вказаним ім'ям. Така помилка виникає
коли якась змінна щє не була ініційована в коді.
t = 55
